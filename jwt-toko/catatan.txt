Catatan untuk menjalankan program : 

1) Menggunakan 
XAMPP 5.6.40
PHP Version 5.6.40

2) Pada "D:\xampp\php\php.ini"
// ubah
short_open_tag=Off
// menjadi
short_open_tag=On

3) Restart apache di XAMPP Control Panel

4) JWT menggunakan library firebase/php-jwt v5.5.1, download di
https://github.com/firebase/php-jwt/archive/refs/tags/v5.5.1.zip
Library firebase/php-jwt v5.5.1 mendukung PHP v5.6 yang kita gunakan. Untuk mendukung PHP yang terbaru, gunakan library firebase/php-jwt yang terbaru.
